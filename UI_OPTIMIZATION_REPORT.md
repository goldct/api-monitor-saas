## 🎨 设计系统

### 配色方案
- **主色**: 蓝色到靛蓝渐变 (`from-blue-600 to-indigo-600`)
- **次色**: 橙色到红色渐变 (`from-orange-500 to-red-500`)
- **背景**: 白到浅蓝渐变 (`from-blue-50 via-white to-indigo-50`)

### 组件样式
- **圆角**: `rounded-xl` (小组件)、`rounded-2xl` (卡片)
- **阴影**: `shadow-lg` (卡片)、`shadow-xl` (hover)
- **过渡**: `transition-all` + `transform` (hover效果)

### 字体样式
- **标题**: `font-bold` + 大尺寸
- **正文**: `font-medium` 或默认
- **代码**: `font-mono` (URL等)

## 📝 技术细节

### 使用的Tailwind特性
- 渐变：`bg-gradient-to-br`
- 背景模糊：`backdrop-blur-sm`
- 阴影：`shadow-lg`, `shadow-xl`
- 过渡：`transition-all`
- 变换：`transform hover:-translate-y-1`

### 动画类
```css
.animate-slide-in {
  animation: slideIn 0.3s ease-out;
}

.animate-scale-in {
  animation: scaleIn 0.2s ease-out;
}
```
